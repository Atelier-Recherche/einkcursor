# Eink Cursor

Plugin Obsidian : curseur d’édition statique (sans animation), pensé pour les écrans e-ink.

## Installation

Copie ce dossier dans le vault :

`.obsidian/plugins/eink-cursor/`

(avec `manifest.json`, `main.js`, `styles.css` à la racine du dossier du plugin.)

Puis **Paramètres → Communauté → Plugins tiers** : activer **Eink Cursor**.

## Utilisation

Aucun réglage : active le plugin, c’est tout.

Ne pas utiliser en parallèle avec un autre plugin qui remplace le curseur (ex. Animated Cursor).
